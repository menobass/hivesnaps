# Debug Scripts

This folder contains debugging and development utility scripts for HiveSnaps.

## Purpose
These scripts are used for:
- Testing specific features in isolation
- Debugging issues during development
- Verifying data formats and parsing logic
- Quick prototyping and experimentation

## Scripts

- `debug-post-type.js` - Debug post type detection and classification
- `debug-spoiler.js` - Test spoiler tag parsing and rendering
- `debug_spoiler.js` - Alternative spoiler tag testing

## Usage
Run scripts with Node.js:
```bash
node debug/debug-post-type.js
```

**Note:** These scripts are for development only and should not be run in production.
