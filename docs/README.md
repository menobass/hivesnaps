# HiveSnaps Website

This directory contains the website files for HiveSnaps.

## GitHub Pages Setup

The website is served from the `/docs` folder and accessible at:
https://menobass.github.io/hivesnaps

## Files

- `index.html` - Main landing page
- `privacy-policy.html` - Privacy policy (required for Google Play Store)

## Local Development

To test locally, you can serve the files using any simple HTTP server:

```bash
# Using Python
cd docs
python -m http.server 8000

# Using Node.js
npx serve docs
```
